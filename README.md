# Portfolio - Muhammad Siraj Habeebur Rahman

Professional portfolio website showcasing full-stack development skills, projects, and technical expertise.

## Overview

Modern, responsive portfolio built with Next.js, featuring a dark mode premium design aesthetic. The site highlights production experience, technical skills, and academic background in software development.

## Tech Stack

- **Framework**: Next.js 15.1.4 (App Router)
- **Language**: TypeScript 5.x
- **Styling**: Custom CSS with design tokens
- **Deployment**: Vercel-ready
- **Fonts**: Custom typography system

## Design Features

- Dark mode premium aesthetic
- Minimal professional interface
- Subtle blue accent color system
- Glassmorphic card components
- Smooth slide animations
- Responsive layout for all devices
- Backdrop blur effects
- Optimized image loading

## Project Structure

```
src/
├── app/
│   ├── globals.css          # Design system and components
│   ├── layout.tsx            # Root layout with fonts
│   ├── page.tsx              # Homepage
│   └── work/[slug]/          # Project detail pages
└── fonts/                    # Custom typography
```

## Key Sections

- **Hero**: Introduction with focus areas
- **Snapshot**: Current role and availability
- **Work**: Featured projects and technical tracks
- **Skills**: Core competencies and technologies
- **Experience**: Education and learning journey
- **Contact**: Professional contact information

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the application.

Build for production:

```bash
npm run build
```

## Development

The site uses custom CSS variables for theming and consistent design patterns across all components. All card components follow a unified dark mode aesthetic with professional hover states and transitions.
